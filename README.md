## Čo to je?

Buzerátor je Slack bot, ktorý sa pravidelne pýta ľudí v kanáli, aby mu napísali.
Vhodný na pravidelné zisťovanie stavu / updatov...

## Konfigurácia

- `SLACK_APP_TOKEN`
- `SLACK_CLIENT_ID`
- `SLACK_CLIENT_SECRET`
- `ROOT_URL`
- `LISTEN_ADDRESS`
- `DATABASE_FILE`
